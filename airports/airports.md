## How to get airport data

The following command was used to grab the airport data hosted at https://data-usdot.opendata.arcgis.com/datasets/airports/geoservice?orderBy=OBJECTID&orderByAsc=false
<br>
```bash
curl https://geo.dot.gov/server/rest/services/NTAD/Airports/MapServer/0/query?where=1%3D1&outFields=*&outSR=4326&f=json > airports.txt
```
